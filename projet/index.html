Les codes relatifs aux cours de GeoCarto sont dans les fichiers suivants :

  • Fonctions helpers (distance, angles...)

  • Permet de récupérer l'orientation du téléphone (Absolue et relatif - Android et iOS)
  • Permet de récupérer la position actuelle de l'appareil

  • Composant Map permettant d'afficher une carte avec un marker et possiblement un trajet
  • La flèche ThreeJS 3D utilisé pour se diriger
  • Le canvas ThreeJS où l'on rend la flèche

  • La page d'exploration qui appelle la caméra et la scène 3D pour se diriger
  • La page de détail sur un spot que l'on a créé (contient une map avec un marker et le trajet pour s'y rendre)
  • La page de détail sur un spot que l'on a découvert (contient une map avec un marker et le trajet pour s'y rendre)
